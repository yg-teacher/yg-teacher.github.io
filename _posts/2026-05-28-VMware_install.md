## 🚀 [개요]
Windows 환경에서 약 1TB 용량 디스크를 7:3 비율로 분할하여  
독립된 환경을 구성하는 방법을 정리합니다.

## 🔍 [VMware 다운로드]
1. https://www.vmware.com에 접근하여 상단 메뉴 중 Products를 클릭 후
   하단에 위치한 SEE DESKTOP HYPERVISORS 버튼을 클릭합니다.
   <img width="1213" height="828" alt="image" src="https://github.com/user-attachments/assets/4bc961eb-4c20-4211-8f7a-4dc5c6f817e0" />

2. "DOWNLOAD NOW" 버튼을 클릭합니다.
   <img width="1195" height="457" alt="image" src="https://github.com/user-attachments/assets/0696ce34-e3c9-46f4-8d61-948cdf2bcfa2" />

3. 우측 "LOGIN" 버튼을 클릭 후 "REGISTER" 버튼을 선택하여 회원가입 후 로그인을 진행합니다.
   <img width="1487" height="641" alt="image" src="https://github.com/user-attachments/assets/eeea68a8-12bf-435d-a775-0f97bd6f25a7" />
